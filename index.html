Networking Virtual Machines

Sean Robinson, Jacob Ward, and Tyler Mumford

Goal:

Create four virtual machines and demonstrate that they can be used as web servers, can be accessed via ssh, and can form a network.

Overview:

  1. We created our virtual machines using DigitalOcean, a hosting company that makes it easy to spin up servers. The developer has full control of the machine, unlike other common hosting providers who create managed environments.
  2. We demonstrate the networking capabilities of the servers by showing that they can ping one another.
  3. We configured one of the servers to host this website that you are viewing now.
  4. Finally, we will show their ssh capabilities and complete the demonstration by having the class connect to the machine and create files for the web server to display.
